# 📊 Superstore Sales Analysis Dashboard

## 🧾 Project Overview
This project is an interactive Power BI dashboard developed using the Superstore dataset. The dashboard provides insights into sales performance, profitability, customer segments, and regional trends, helping businesses make data-driven decisions.

## 🎯 Objectives
- Analyze overall sales and profit performance.
- Identify top-performing products and categories.
- Compare sales and profit across different regions.
- Understand customer segment contributions.
- Monitor monthly sales trends and business growth.

## 🛠️ Tools & Technologies
- Power BI
- Microsoft Excel / CSV Dataset
- Power Query
- DAX (Data Analysis Expressions)

## 📈 Dashboard Features
### Executive Summary
- Total Sales
- Total Profit
- Total Orders
- Average Sales per Order

### Sales Analysis
- Sales by Category
- Sales by Sub-Category
- Top Performing Products

### Profit Analysis
- Profit by Category
- Profit by Region
- Loss-Making Products Identification

### Customer Analysis
- Sales by Customer Segment
- Segment-wise Profit Contribution

### Regional Analysis
- Sales by Region
- Profit by Region
- Geographic Performance Comparison

### Time Series Analysis
- Monthly Sales Trend
- Monthly Profit Trend
- Year-over-Year Performance Tracking

## 📂 Project Files
- `Superstore Project.pbix` – Power BI Dashboard File
- `Dataset.xlsx/CSV` – Source Dataset
- `README.md` – Project Documentation

## 📊 Key Insights
- Identified top-performing product categories.
- Analyzed regional sales and profit distribution.
- Tracked business performance over time.
- Highlighted customer segments contributing the highest revenue.
- Discovered opportunities for improving profitability.

## 🚀 How to Use
1. Download the `.pbix` file.
2. Open it using Power BI Desktop.
3. Refresh the dataset if required.
4. Interact with filters and visuals to explore insights.
